SM1 - A Corrida do Prompt
Descrição
Este projeto documenta testes de engenharia de prompt realizados para a geração de imagens (estilo barroco) e redação formal. O objetivo foi refinar a interação com a IA para obter resultados artísticos e contextuais específicos.

Metodologia
Iteração: Testes sucessivos refinando o estilo artístico (Chiaroscuro, textura de tela, craquelado).
Contextualização: Criação de prompts para escrita de e-mails formais (ex: pirata para rei), focando em tom e estrutura.
Resultados
Ajuste fino de prompts resultou em imagens com estética de pintura a óleo envelhecida, superando composições digitais simples.
