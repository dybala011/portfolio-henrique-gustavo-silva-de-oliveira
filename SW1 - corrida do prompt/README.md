# Projeto: Refinamento de Prompts e Escrita Criativa

Este repositório documenta a evolução de comandos (prompts) para geração de imagens via IA, demonstrando como o ajuste de detalhes técnicos e artísticos influencia o resultado final. Além disso, inclui um exemplo de redação criativa aplicada a um contexto formal fictício.

## 🚀 Evolução da Engenharia de Prompt

O objetivo principal foi gerar a imagem de um **astronauta tocando violoncelo em Marte**, evoluindo de uma estética moderna para o estilo **Barroco do século XVII**.

### 📈 Ciclo de Refinamento

| Versão | Descrição do Prompt | Avaliação do Resultado |
| :--- | :--- | :--- |
| **01** | [cite_start]"Gere uma imagem tocando violoncelo em Marte" [cite: 4] | [cite_start]Estilo realista simples, iluminação comum e pouca dramaticidade. [cite: 6, 9, 10] |
| **02** | [cite_start]Adição de estilo barroco e iluminação *chiaroscuro*. [cite: 19, 20] | [cite_start]Melhor iluminação, mas o traje espacial ainda era muito futurista. [cite: 23, 26] |
| **03** | [cite_start]Inclusão de "pinceladas dramáticas", "traje ornamentado em dourado" e "céu tempestuoso". [cite: 35, 36] | [cite_start]Estética barroca clara, porém com acabamento digital "limpo" demais. [cite: 42, 44] |
| **04** | [cite_start]Adição de "textura de tela", "craquelado" (rachaduras) e inspiração em Caravaggio. [cite: 51, 52] | [cite_start]**Resultado Ideal:** Aparência de pintura antiga com forte dramaticidade emocional. [cite: 60, 62, 63] |

## 🎨 Elementos Técnicos Utilizados

Para alcançar a integração artística desejada, foram aplicados os seguintes conceitos:
* [cite_start]**Chiaroscuro:** Contraste forte entre luz e sombra para criar volume e drama[cite: 15].
* [cite_start]**Craquelado:** Simulação de rachaduras finas que surgem em pinturas a óleo antigas com o tempo[cite: 48, 56].
* [cite_start]**Ornamentação Barroca:** Detalhes dourados e complexos no traje para quebrar a estética tecnológica moderna[cite: 17, 31].
